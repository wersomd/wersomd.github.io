# Серьёзный редизайн + контент — Design Spec

**Date:** 2026-07-21
**Owner:** wersomd (serdivergent@gmail.com)
**Site:** wersomd.github.io (GitHub Pages, static, RU)

## Goal
Смягчить neon-cyberpunk тему сайта до более сдержанного, «дорогого» tech-стиля
(в духе Apple), сохранив акцентные цвета cyan/magenta. Плюс — расширить портфолио
новыми проектами и заменить отзывы на реалистичные.

## Design tone
- Оставляем: тёмный фон, cyan/magenta акценты, шрифты (Space Grotesk / Inter / IBM Plex Mono).
- Убираем: анимацию glitch-текста (`.glitch` keyframes), слой `.scanlines`,
  повсеместный `text-shadow`/`box-shadow` неоновый glow на заголовках, карточках,
  логотипе, кнопках, `pulse-dot` пульсацию.
- Усиливаем: воздух между блоками, спокойный hover (лёгкий transform/тень вместо
  свечения), свечение оставляем точечно (например только primary CTA), кнопки —
  залитый уверенный цвет вместо неоновой обводки.
- HTML/структура секций не меняется — правки только в `styles/main.css` (плюс
  удаление связанных с glitch/scanlines фрагментов разметки и JS, если они не
  нужны без эффекта).

## Портфолио — изменения в группе «Сайты и магазины»
Текущие 3 позиции → 8. Добавляем (после Comfort Mebel):

1. **Comfort Mebel** (comfortmebel.kz) — мета меняем на «Премиум-мебель».
2. **Attila** (attilla.tilda.ws) — «Премиум барбершоп» (Шымкент), ссылка активна.
3. **Ummaisha** (ummaishaa.tilda.ws) — «Кондитерская — каталог», ссылка активна.
4. **CIP EDU** (cipedu.kz) — «Образовательный центр» — без ссылки (хостинг отключён),
   оформление как текстовые пункты в блоках Bitrix24/amoCRM (без `<a>`).
5. **MAF Park** (maf-park.kz) — «Игровые площадки» — без ссылки (хостинг отключён).
6. **Kerege Group — ЖК** — «Сайт для продажи ЖК» — без ссылки (ещё не в
   продакшене), без статусных пометок.

`proj-count` для группы обновляется с «03» на «08».

## Отзывы — замена всех трёх
1. **Бауыржан / Comfort Mebel** — реальный текст клиента (система учёта +
   поддержка после сдачи), используется как есть, слегка адаптирован под формат
   карточки.
2. **Абай / 961.kz** — переписывается в живом разговорном тоне, конкретика по
   интернет-магазину + amoCRM + телефония + склад.
3. **Led-Steppe** — переписывается аналогично, конкретика по внедрению Bitrix24.

## Out of scope
- Изменение структуры секций/навигации.
- Смена шрифтов и цветовой палитры.
- Новые отзывы под новые проекты (только эти три).
- Индикация статуса Kerege Group ЖК (по решению заказчика — без пометок).
